# 小许打字 (XTT) - 单文件打字练习应用

一个完全自包含的打字练习Web应用，支持英文打字、五笔单字和五笔词组练习。

## 核心功能

### 🌐 完全独立
- 所有数据内嵌在单个HTML文件中
- 无需网络连接，无需外部文件
- 开箱即用，下载即运行

### 🎯 三种练习模式
1. **英文打字练习** - 100篇精选英文文章
2. **五笔单字练习** - 2,027个常用汉字五笔编码
3. **五笔词组练习** - 1,000个常用词组五笔编码

### 📊 实时数据统计
- 打字速度（字/分钟）
- 准确率百分比
- 用时统计
- 进度跟踪

### 🎨 现代化界面
- 简洁美观的UI设计
- 可爱的装饰元素
- 响应式布局
- 友好的错误提示

### 🔧 数据可更新
- 通过脚本更新五笔编码数据
- 通过脚本更新练习内容
- 支持重复执行，不会产生重复数据

## 快速使用

### 直接使用
```bash
# 下载并打开 all_in_one.html
# 无需任何其他步骤
```

### 更新数据（可选）
```bash
# 更新五笔数据
node update_wubi.js

# 更新练习内容
node update_content.js
```

## 技术特点

### 数据嵌入
- 所有数据压缩为单行JSON
- 通过JavaScript变量内嵌在HTML中
- 无网络请求，加载速度快

### 智能更新
- 自动识别已有数据并替换
- 支持无限次重复执行
- 保持向后兼容性

### 文件大小
- 主文件仅约 0.28 MB
- 包含所有练习内容和五笔编码

## 浏览器支持
- Chrome / Firefox / Safari / Edge 等现代浏览器
